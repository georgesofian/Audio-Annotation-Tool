Anno (Audio Annotation Tool)

Acest raport descrie curentul progress la proiectul numit „Anno (Audio Annotation Tool)” care va fi evaluat la disciplina Dezvoltarea Aplicatiilor Web la Nivel de Cleint promoția 2017-2018. Acest proiect a fost realizat de studentul Sofian George-Andrei,anul III, grupa B7.

1.Introducere

Se doreşte implementarea unui instrument Web (eventual, o extensie de navigator sau componentă Web) pentru adnotarea conţinuturilor audio redate utilizatorului. O adnotare poate include termeni de conţinut (tag-uri) sau poate avea ataşate resurse specifice: imagini, alte melodii, videoclip-uri, coordonate geografice, persoane ce au contribuit la acel conţinut (de exemplu, compozitor sau interpret) etc. Sistemul va putea realiza managementul tuturor adnotărilor specificate de un utilizator cu posibilitatea exportării acestora în formate CSV, JSON şi XML.

1. Etapele dezvoltarii proiectului

  1. Schite;
  2. Creare pagini popup;
  3. Design & implementare;
  4. Implementarea pe baza html/css/javascript;

2.Tehnologi utilizate

Proiectul utilizează următoarele tehnologii:
  1. HTML5 - este un limbaj pentru structurarea și prezentarea conținutului pentru World Wide Web, o tehnologie nucleu pentru Internet propusă inițial pentru software-ul Opera.
  2. CSS - (Cascading Style Sheets) este un standard pentru formatarea elementelor unui document HTML. Stilurile se pot atașa elementelor HTML prin intermediul unor fișiere externe sau în cadrul documentului, prin elementul "style" și/sau atributul style. CSS se poate utiliza și pentru formatarea elementelor XHTML, XML și SVGL
  3. Javascript - este un limbaj de programare orientat obiect bazat pe conceptul prototipurilor.Este folosit mai ales pentru introducerea unor funcționalități în paginile web, codul Javascript din aceste pagini fiind rulat de către browser. Limbajul este binecunoscut pentru folosirea sa în construirea siturilor web, dar este folosit și pentru acesul la obiecte încastrate (embedded objects) în alte aplicații. A fost dezvoltat inițial de către Brendan Eich de la Netscape Communications Corporation sub numele de Mocha, apoi LiveScript, și denumit în final JavaScript.
    În ciuda numelui și a unor similarități în sintaxă, între JavaScript și limbajul Java nu există nicio legătură. Ca și Java, JavaScript are o sintaxă apropiată de cea a limbajului C, dar are mai multe în comun cu limbajul Self decât cu Java.
  4. Api-ul Chrome a fost folosit pentru a prelua informatiile necesare din tabul curent si pentru a le salva. Comunicarea dintre popup si tab s-a facut pe baza mesajelor. Aceasta comunicare a fost folosita pentru extragerea informatiilor din paginile care contin video-uri. Recunoastem o pagina care contine video prin intermediul valorii campului "time" din raspuns. Chrome.storage a fost folosit pentru a salva datele si a le reda ulterior in tab-ul corespunzator.

3. Structura

Pagina "popup.html" contine:
  1. Un div care contine URL-ul tabului si unul care contine secunda curenta la care se afla video-ul din pagina respectiva(daca acesta exista,altfel va fi null);
  2. Un buton "Get Info" care va seta URL-ul si timpul;
  3. Campuri pentru introducerea adnotarilor(Artists,Tags,Links);
  4. Un buton "Save" care va salva adnotarile;
  5. Un buton "Reset page" care va sterge adnotarile din pagina curenta;
  6. Un buton "Export" care va exporta adnotarile.
Extensia va insera in tab un div care va contine:
  1. Un camp pentru afisarea adnotarilor de tip "Artist";
  2. Un camp pentru afisarea adnotarilor de tip "Tag";
  3. Un camp pentru afisarea adnotarilor de tip "Link".
